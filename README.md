# Layout Ventures Website

Official website for Layout Ventures - backing first institutional rounds for companies selling into healthcare.

## Overview

Layout Ventures backs any innovative technology that makes healthcare cheaper, better, faster, or more accessible — whether it's clinical or behind the scenes.

## Website Structure

- **index.html** - Main homepage with hero section, approach pillars, portfolio, and team
- **fund-announcement.html** - Fund announcement page
- **styles.css** - Main stylesheet
- **announcement-styles.css** - Additional styles for announcement page
- **script.js** - JavaScript for smooth scrolling and animations

## Features

- Responsive design for all devices
- Smooth scrolling navigation
- Portfolio showcase with hover effects
- Team section with social links
- Clean, professional design with custom color scheme

## Color Scheme

- Primary Green: `#00a3b8`
- Dark Green: `#085466`
- White: `#ffffff`
- Black: `#000000`
- Light Gray: `#eff1f3`
- Text Gray: `#333333`

## Setup

1. Clone this repository
2. Open `index.html` in a web browser
3. No build process or dependencies required - pure HTML, CSS, and JavaScript

## Deployment

This is a static website and can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service
